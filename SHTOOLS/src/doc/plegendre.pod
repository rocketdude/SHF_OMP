=head1 PLegendre

=over 12

=item PLegendre -  

Compute all the (unnormalized) Legendre polynomials.

=back

=head1 SYNOPSIS

=over 23

=item SUBROUTINE PLegendre ( 

P, LMAX, Z )

=back

=over 4

=over 10

=item REAL*8	

P(LMAX+1), Z

=item INTEGER

LMAX

=back

=back

=head1 DESCRIPTION

I<PLegendre> will calculate all of the (unnormalized) Legendre polynomials up to degree LMAX for a given argument. These are calculated using a standard three-term recursion formula. The integral of the Legendre polynomials over the interval [-1, 1] is 2/(2L+1).

=head1 ARGUMENTS

=over 10

=item P

(output) REAL*8, DIMENSION (LMAX+1)

An array of (unnormalized) Legendre polynomials up to degree LMAX. Degree L corresponds to array index L+1.
	
=item LMAX

(input) INTEGER

The maximum degree of the Legendre polynomials to be computed.

=item Z

(input) REAL*8

The argument of the Legendre polynomial.

=back
