=head1 PlmIndex

=over 11

=item PlmIndex -  

Compute the index of an array corresponding to degree L and angular order M.

=back

=head1 SYNOPSIS

=over 28

=item INTEGER FUNCTION PlmIndex ( 

L, M )

=back

=over 4

=over 10

=item INTEGER

L, M

=back

=back

=head1 DESCRIPTION

I<PlmIndex> will calculate the index of an array corresponding to degree L and angular order M. The input arrays are generated by routines such as I<PlmBar>, I<PlmBar_d1>, I<PlmSchmidt>, I<PlmSchmidt_d1>, I<PlmON>, I<PlmON_d1>, I<PLegendreA>, and I<PLegendreA_d1>. The array index is equal to L*(L+1)/2+M+1.

=head1 ARGUMENTS

=over 10

=item L

(input) INTEGER

The spherical harmonic degree.
	
=item M

(input) INTEGER

The angular order.

=back
